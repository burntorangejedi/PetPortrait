# PetPortrait

A World of Warcraft retail addon (version 11.2.7) that extends the Pet Journal with an "Enlarge" button. When clicked, it opens a resizable window displaying a large 3D model of the selected pet.

## Features

- **Pet Portrait Window**: View an enlarged 3D model of your battle pets
- **Resizable Window**: Drag to resize and reposition the window
- **Live Updates**: Switch between pets in the Journal to instantly update the portrait
- **Easy Access**: "Enlarge" button located next to the Summon button in the Pet Journal

## Installation

1. Download the addon folder
2. Place it in your WoW AddOns directory:
   ```
   World of Warcraft\_retail_\Interface\AddOns\PetPortrait
   ```
3. Enable the addon in the Addons list
4. Open the Pet Journal and click "Enlarge" to view a pet's portrait

## Usage

1. Open the Pet Journal (default: `P` key)
2. Select a pet from your collection
3. Click the "Enlarge" button next to the Summon button
4. A window will open displaying the pet's 3D model
5. Resize the window by dragging the edges
6. Close the window with the X button or select a different pet to update the display

## Dependencies

- Ace3 (LibStub, AceAddon-3.0, AceGUI-3.0)

## Requirements

- World of Warcraft Retail (version 11.2.7 or higher)
